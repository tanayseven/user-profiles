User Profiles
=============


Commands
--------

Install dependencies
```bash
npm install
```

Run in dev mode
```bash
npm start
```

Create a production build
```bash
npm run build
```

Run all the tests
```bash
npm test
```

Run mutation tests
```bash
npm run test-mutation
```

Run e2e or ui tests
```bash
npm run test-ui
```

Run linter
```bash
npm run lint
```

Fix fixable linter rules
```bash
npm run lint:fix
```
